# **App Name**: Risabu Connect ERP

## Core Features:

- Admin Role-Based Authentication: Secure login and user role management for administrators, staff, and potentially students, utilizing Firebase Authentication.
- Student Admission Management: System to manage student applications, track admission status, and store student profiles. Includes basic data capture for enrollment.
- Financial Invoice & Payment Tracking: Ability to generate tuition and fee invoices, record student payments, and track outstanding balances. Uses Firestore for data storage.
- College Expense Management: Module for recording, categorizing, and tracking institutional expenses. All data stored in Firestore.
- Financial Overview Dashboard: A centralized dashboard providing key financial summaries, payment statuses, and expense breakdowns for quick insights.
- AI-Powered Financial Insights Tool: Utilizes AI to analyze recorded financial data, identify spending trends, highlight unusual expenses, or predict cash flow patterns to assist financial planning.

## Style Guidelines:

- Primary color: A deep, professional green (#2E8B57) symbolizing growth, stability, and education, appropriate for an educational and financial institution.
- Background color: A very light, subtle green-gray (#E8F5E9), derived from the primary hue but heavily desaturated, providing a clean and non-distracting canvas.
- Accent color: A vibrant, energetic lime green (#66BB6A) analogous to the primary, used sparingly for interactive elements like buttons and notifications to draw attention effectively.
- Body and headline font: 'Inter' (sans-serif), chosen for its clean, modern, and highly legible characteristics, essential for an information-rich ERP system.
- Use clear, universally recognizable icons with a minimalist, line-art style. Icons should represent common ERP functions like 'add student', 'generate invoice', 'view reports' effectively.
- Adopt a structured, grid-based layout with ample whitespace to ensure readability and easy navigation through data-dense interfaces. Dashboards will feature cards for quick summary views.
- Implement subtle and functional animations, such as smooth transitions for data loading or form submissions, to provide positive user feedback without distracting from tasks.
